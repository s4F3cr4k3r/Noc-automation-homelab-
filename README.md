Windows NOC Automation Lab
This project simulates a small-scale Network Operations Center (NOC) environment using automated monitoring and remote remediation.
Using PRTG, PowerShell, and Windows-based infrastructure, I built a system capable of:

1-Detecting CPU and RAM threshold violations
2-Automatically restarting remote systems
3-Monitoring critical Windows services and auto-restarting them
4-Logging every action with timestamps
5-Generating daily reports and email alerts


Instead of passive monitoring, this lab implements a full automation cycle:
Detect → Trigger → Remediate → Log → Report
This project demonstrates practical NOC-style operations and proactive infrastructure management.
